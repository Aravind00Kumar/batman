List Component Example

Installation and usage

  • Add the following scripts in index.html file
    <script src="./node_modules/@batman/es5-components/list.js">
  • In index.html add the following code
    <div class="panel flex item-full column no-mar-collapse">
    <div id="l1" class="flex item-full">
    </div>
  • In script.js add the following jscode
    document.addEventListener("DOMContentLoaded", function () {
    var dataSource = [{ "text": "0" }, { "text": "1" }, { "text": "2" }];
    var List = window.Batman.list.List;
    new List(document.getElementById("li"), {
    data: dataSource,
    height: 30,
    pageSize: 5
    });
    });
