Native JavaScript components installation and usage

  • Install the packages
    npm install ..\\build\\batman-es5-1.0.0-alpha.2.tgz
  • Add the following scripts in index.html file
    <script src="./node_modules/@batman/es5-components/polyfills.js">
    <script src="./node_modules/@batman/es5-components/core.js">
  • Add individual component files in index.html
    <script src="./node_modules/@batman/es5-components/{component}.js">
    {component} : list , tree and etc
