Angularjs Wrappers

angularjs components installation and usage

  • Install the packages
    npm install ..\\build\\batman-core-es5-1.0.0-alpha.2.tgz
    npm install ..\\build\\batman-components-es5-1.0.0-alpha.2.tgz
    npm install ..\\build\\batman-ajs-1.0.0-alpha.2.tgz
  • Add the following scripts and styles in index.html file
    <link rel="stylesheet" href="./node_modules/@batman/core-es5/batman.css">
    <script src="./node_modules/@batman/core-es5/polyfills.js">
    <script src="./node_modules/@batman/core-es5/core.js">
    <script src="./node_modules/@batman/components-es5/doughnut.js">
    <script src="./node_modules/@batman/components-es5/list.js">
    <script src="./node_modules/@batman/components-es5/tree.js">
    <script src="./node_modules/angular/angular.js">
    <script src="./node_modules/@batman/ajs/module.js">
    <script src="./node_modules/@batman/ajs/list.js">
  • Add individual component files in index.html
    <script src="./node_modules/@batman/components-es5/{component}.js">
    <script src="./node_modules/@batman/ajs/{component}.js">
    {component} : list , tree and etc
